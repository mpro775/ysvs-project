# بذور البيانات (Database Seeds)

بيانات وهمية كاملة لنظام الجمعية اليمنية للجراحة الوعائية (YSVS).

## التشغيل

```bash
# إضافة البيانات الوهمية (بدون حذف البيانات الحالية)
npm run seed

# حذف كل البيانات ثم إعادة البذر من الصفر
npm run seed:fresh
```

## المتطلبات

- MongoDB يعمل ومتصل (مثلاً `mongodb://localhost:27017/ysvs` أو عبر `MONGODB_URI` في `.env`).

## الخيارات

- **`--fresh`**: يحذف كل البيانات في المجموعات ذات الصلة ثم يعيد البذر من البداية. استخدمه عند الرغبة في قاعدة نظيفة.
